### Mark Hermes

GNC (Guidance, Navigation & Control) engineer working in the space sector. PhD in Mechanical Engineering. I build flight software that takes vehicles from orbit to the ground: state estimation, guidance, and control, validated in simulation and flight test.

**Focus areas**
- Navigation and state estimation (Kalman filtering, sensor fusion)
- Guidance and trajectory optimization (reentry, atmospheric, optimal control)
- Attitude determination and control (reaction wheels, thrusters, allocation)
- Real-time embedded flight software in C/C++ and Python

**Tools I work in:** C, C++, Python, MATLAB, Basilisk, GMAT, POST2, EarthGRAM, IPOPT/PSOPT

Much of my production flight work is proprietary. The repositories below are personal and academic projects that show how I approach estimation, control, and trajectory optimization.
